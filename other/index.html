Welcome to hh.other
This is my a treasure trove full of things that aren't important enough to put on the navbar.
hh.old-home is a page that has the old homepage.
hh.tools is a directory that houses usefull tools.
hh.layouts is a directory that has all layouts in it.
hh.fonts is a directory that houses the font files, and it houses a page showing the different fonts.
hh.searx-redirect is a page that redirects the visitor to a not blocked instance (not finished).
hh.debug is a directory that houses tools for testing the site.
hh.changelog is a page that houses the changelog. It is titled CHANGELOG.md in the source.
robots.txt is a set of instructions for bots. For more information, please visit https://www.robotstxt.org/
humans.txt is a file that list people involved in the creation of the site. For more information, please visit https://humanstxt.org/
sitemap.xml is a file that tells search engines what should be indexed. For more information, please visit https://www.sitemaps.org/
